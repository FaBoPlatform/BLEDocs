# Keil MDK-ARM

## 現在のバージョン

* 5.16a

## MDK-ARM価格一覧

| ライセンス | 価格 | 詳細 |
| -- | -- | -- |
| MDK-Professional | 30-40万円(1年ライセンス) | Middleware Librariesも付属 |
| MDK-Standard | 約25-30万円(1年ライセンス)| Cortex®-Mシリーズ, ARM7,ARM9,Cortex®-R, SecureCore　|
| MDK-Cortex-M | 約20万円(1年ライセンス）| Cortex®-Mシリーズ限定 |
| MDK-Lite | 無料| 32K制限 |

## Keil MDK-ARMがサポートしているモジュール
